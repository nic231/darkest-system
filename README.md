# The Darkest System — Foundry VTT

A Foundry VTT game system implementing the mechanics for **The Darkest Woods** and **The Darkest House** by Monte Cook Games.

> **Alpha Release** — This system is in active development. Expect rough edges. Feedback welcome.

---

## What's Included

This system provides the mechanical framework only:

- Rating-based task resolution (2d6 action rolls)
- The Darkest Die and Transgression tracking
- Boons and Banes
- Wounds (physical and mental), unconsciousness, death and catatonia checks
- Doom system
- Damage rolls and NPC tracker
- Character, NPC, and Creature sheets
- Darkest Woods and Darkest House game modes

**No game content is included.** Witch names, region descriptions, transgression events, and house actions are not part of this system. You must own the source material (The Darkest Woods or The Darkest House by Monte Cook Games) to use this system meaningfully.

---

## Installation

1. In Foundry VTT, go to **Setup → Game Systems → Install System**
2. Paste the manifest URL into the field at the bottom:
   ```
   https://raw.githubusercontent.com/nic231/darkest-system/master/system.json
   ```
3. Click Install

---

## Requirements

- Foundry VTT v11 or later (verified on v13)
- The Darkest Woods or The Darkest House (purchased from Monte Cook Games)
